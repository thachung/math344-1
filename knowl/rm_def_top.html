Remark 1.1.2. Equivalent alternative to axioms (1)-(3).

Using induction, we can show that if \(\mathcal{T}\) satisfies axiom (3) for any two elements of \(\mathcal{T}\text{,}\) then it satisfies it for any \(n\) elements, for \(n\) arbitrary. In other words if we replace axiom (3) with
(3')
If \(U_1, U_2\in \mathcal{T}\text{,}\) then \(U=U_1\cap U_2\in \mathcal{T}\)
, we get an equivalent definition of a toplogical space. When verifying whether a given \((X,\mathcal{T})\) is a topological space it may be more convenient to use axiom (3') as opposed to axiom (3).
