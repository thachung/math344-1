For any nonnegative numbers \(a_1,a_2,\dots, a_n\) we have
\begin{equation*} \max\{a_i\}_{i=1}^n\leq \sqrt{a_1^2+a_2^2+\cdots a_n^2}\leq \sqrt{n}\max\{a_i\}_{i=1}^n\text{.} \end{equation*}
From this it follows that for any \(x\in \R^n\) and any \(\epsilon > 0\) we have
\begin{equation} B_{\operatorname{box},\sqrt{n}\epsilon}\subseteq B_{\operatorname{euc},\epsilon}(x)\subseteq B_{\operatorname{box},\epsilon}(x)\text{,}\tag{1.3.3} \end{equation}
showing that within any Euclidean metric ball centered at \(\boldx\) we can find a box metric ball centered at \(\boldx\text{,}\) and vice versa. Now given any element \(y\) in a Euclidean metric ball \(B\text{,}\) there is a Euclidean metric ball \(B'\) centered at \(y\) contained in \(B\) by LemmaÂ 1.3.5. By (1.3.3) there is box metric ball \(B''\) centered at \(y\) satisfying \(B''\subseteq B'\subseteq B\text{.}\) Using our covering principle, we conclude that any Euclidean metric ball is covered by box metric balls, and thus \(\mathcal{T}_{\operatorname{euc}}\subseteq \mathcal{T}_{\operatorname{box}}\text{.}\) A very similar argument, using the other inclusion of (1.3.3), shows that \(\mathcal{T}_{\operatorname{box}}\subseteq \mathcal{T}_{\operatorname{euc}}\text{.}\)
