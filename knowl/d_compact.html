Definition 1.15.2. Compact space.

A topologial space \(X\) is compact if every open covering of \(X\) has a finite subcovering: i.e., if \(X=\bigcup_{i\in I}U_i\) is an open covering, then there is a finite set \(\{i_1,i_2,\dots,i_n\}\subseteq I\) such that \(X=\bigcup_{j=1}^n U_{i_j}\text{.}\)
A subset \(Y\subseteq X\) is compact, if it is compact with respect to the subspace topology. Equivalently, \(Y\) is compact if any covering \(Y\subseteq \bigcup_{i\in I} U_i\) by open sets of \(X\) has a finite subcovering.
