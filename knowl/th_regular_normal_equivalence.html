Theorem 1.19.8. Regular and normal equivalences.

Let \(X\) be a \(T_1\)-space.
  1. \(X\) is regular if and only if for all \(x\in X\) and open set \(U\ni x\text{,}\) there is an open set \(V\ni x\) such that \(\overline{V}\subseteq U\text{.}\)
  2. \(X\) is normal if and only if for any closed set \(A\) and open set \(U\) containing \(A\text{,}\) there is an open set \(V\) containing \(A\) satisfying \(\overline{V}\subseteq U\text{.}\)
