Assume \(x\in \overline{A}\text{.}\) For all open sets \(U\) containing \(x\text{,}\) choose \(x_U\in U\cap A\text{.}\) Let \(I\) be the set of all open neighborhoods of \(x\text{,}\) considered as a directed set with reverse inclusion. Define \(f\colon I\rightarrow A\) as \(f(U)=x_U\text{.}\) The net \((x_U)_{U\in I}\) converges to \(x\text{.}\) Indeed, for any open set \(V\) containing \(x\text{,}\) we set \(i_0=V\in I\text{:}\) if \(U\geq V\text{,}\) then \(U\subseteq V\) (by definition of our ordering), and hence \(x_U\in U\subseteq V\text{.}\)
