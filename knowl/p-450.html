  1. Suppose \(f_n\rightarrow f\) in either topology. Fix \(i\in I\) and let \(U_i\ni f(i)\) be any open set containing \(f(i)\in X_i\text{.}\) The set \(U=U_i\times \prod_{j\ne i}U_j\) is an open neighborhood of \(f\) in both topologies. By convergence there is an \(N\) such that \(f_n\in U\) for all \(n\geq N\text{.}\) It follows that \(f_n(i)\in U_i\) for all \(n\geq N\text{.}\) This proves that \(f_n(i)\rightarrow f(i)\) for all \(i\in I\text{.}\)
  2. Suppose the sequence \((f_n)\) converges to \(f\) pointwise. To show \(f_n\rightarrow f\) in the product topology it is enough to show that for any base open set of the form \(U_{i_1}\times U_{i_2}\times\cdots \times U_{i_m}\times \prod_{k\ne i_k }X_k\) there is an \(N\) such that if \(n\geq N\text{,}\) then \(f_n\in U\text{.}\) Since \((f_n)\) converges to \(f\) pointwise, for each \(i_j\text{,}\) \(1\leq j\leq m\) there is an \(N_{i_j}\) such that if \(n\geq N_{i_j}\text{,}\) then \(f_n(i_j)\in U_{i_j}\text{.}\) Setting \(N=\max\{ N_{i_j}\}_{j=1}^m\text{,}\) it follows that if \(n\geq N\) then \(f_n(i_j)\in U_{i_j}\) for \(1\leq j\leq m\text{.}\) Since \(f_n(j)\in X_j\) for all \(j\ne i_j\) we conclude that \(f_n\in U\) for all \(n\geq N\text{,}\) as desired.
