Theorem 1.24.7. Nets and topology.

Let \(X\) be a topological space.
  1. The closure \(\overline{A}\) of any set \(A\) is the set of all \(x\in X\) for which there is a net \(f\colon I\rightarrow A\) such that \(x_i\rightarrow x\text{.}\)
    It follows that a subset \(A\) is closed if and only if it contains all limits of all convergent nets in \(A\text{.}\)
  2. A function \(f\colon X\rightarrow Y\) is continuous if and only if for all convergent nets \(h\colon I\rightarrow X\) the net \(f\circ h\colon I\rightarrow Y\) is convergent.
  3. The space \(X\) is Hausdorff if and only if every net in \(X\) converges to at most one element in \(X\text{.}\)

Proof.
