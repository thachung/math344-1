1. Consider the map \(g\) defined by composition
    where \(q\) is the quotient map from \(\R\) to \(\R/\Z\text{.}\) It is easy to see that \(g\) is surjective and satisfies \(g(t)=g(t')\) if and only if \(t=t'\) or \(t,t'\in \{0,1\}\text{.}\) From Corollary 1.11.7 we conclude that \(g\) gives rise to a bijective continuous function \(f\colon X^*\rightarrow \R/\Z\text{.}\) Concretely, \(f\) takes an equivalence class \([t]\in X^*\) and maps it to the equivalence class \(\{t+n\colon n\in \Z\}\) in \(\R/\Z\text{.}\) Furthermore, to show \(f\) is a homeomorphism it is enough to prove that \(g\colon [0,1]\rightarrow \R/\Z\) is a quotient map. In fact, we can show that \(g\) is a closed map. Take a closed set \(C\subseteq [0,1]\text{.}\) To show its image \(g(C)\) is closed in \(\R/Z\text{,}\) we need only show, using the fact that \(q\) is a quotient map, that \(q^{-1}(g(C))\) is closed in \(\R/\Z\text{.}\) We have \(q^{-1}(g(C))=C+\Z=\{c+n\colon c\in C, n\in \Z\}\text{.}\) This set is the result of starting with our original closed set \(C\subseteq [0,1]\) and taking the union of translated copies of the form \(C+n\subseteq [n,n+1]\text{,}\) where \(n\in \Z\text{.}\) It is easy to see that each translate \(C+n\) is closed; since furthermore \(C+n\subseteq [n,n+1]\text{,}\) it follows that the union is also closed.     where \(g(t)=(\cos 2\pi t, \sin 2\pi t)\text{.}\) It is easy to see that \(g\) is continuous, surjective, and satisfies \(g(t_1)=g(t_2)\iff t_1=t_2+n\) for some \(n\in \Z\text{.}\) It follows from Corollary 1.11.7 that the map \(f\) is bijective and continuous. Furthermore, by the same result, we can show \(f\) is a homeomorphism as long as we can show \(g\) is a quotient map. I'll show that in fact \(g\colon \R\rightarrow S^1\) is open, and hence a projection map.
    To this end let \(U\subseteq \R\) be any open set, and let \(P\in f(U)\text{,}\) so that \(P=g(t)\) for some \(t\in U\text{.}\) We will find an open interval \(I\) satisfying \(t\in I\subseteq U\) such that \(f(I)\) is open. Since \(P\in f(I)\subseteq f(U)\) and \(P\in f(U)\) was arbitrary, this will show that \(f(U)\) is open.
    Let \(g(t)=P=(x,y)\text{.}\) Since we cannot have \(x=y=0\text{,}\) we have four cases: \(y>0\) or \(y< 0\) or \(x> 0\) or \(x< 0\text{.}\) I will treat the first case; my argument is easily adapted to the other cases. Assume \(y> 0\text{.}\) Since the set \(V=S^1\cap \left(\R\times (0,\infty)\right)\) is open in \(S^1\text{,}\) we can find an open interval \(I=(a,b)\ni t\) contained in \(U\) such \(g(I)\subseteq V \text{:}\) i.e., all points in \(g(I)\) lie in the the upper half plane. Elementary trigonometric properties tell us that the function \(h(t)=\cos 2\pi t\) is strictly decreasing on \(I\text{,}\) and hence that \(h(I)\) is the open interval \((\cos 2\pi b, \cos 2\pi a)=(c,d)\text{.}\) It is now easy to see that \(g(I)=\{(x,y)\colon x\in (c,d), y=+\sqrt{1-x^2}\}=S^1\cap \left((c,d)\times \R\right)\text{,}\) which is open.
