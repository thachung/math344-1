Topological specimen 14. Topologist's sine curve.

Define \(f\colon (0,1]\rightarrow \R\) as \(f(t)=\sin(1/t)\) for all \(t\in (0,1)\text{.}\) Let \(\Gamma_f=\{(t,f(t))\in \R^2\colon t\in (0,1)\}\text{,}\) the graph of \(f\text{.}\)
  1. The set \(X=\overline{\Gamma_f}=\Gamma_f\cup \{0\}\times [0,1]\) is connected.
  2. The set \(X\) is not path connected. More specifically, we can show that \((0,0)\) and \((1,\sin(1))\) are not connected by a path in \(X\text{.}\)
