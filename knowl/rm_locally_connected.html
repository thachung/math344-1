Remark 1.14.4. Connected does not imply locally connected.

Somewhat curiously, a space being connected does not necessarily imply that the space is locally connected. For example, the topologist's sine curve \(\overline{S}\) is connected, but not locally connected. For example, take the point \(P=(0,0)\in \overline{S}\) and the open set \(U=[0,1]\times (-1/2,1/2)\cap \overline{S}\text{.}\) Graphically, it is easy to see that \(U\) has a connected component partition consisting of \(\{0\}\times (-1/2,1/2)\) and infinitely many connected segments of the graph of \(f(x)=\sin (1/x)\) (one segment for each interval of the form \((1/(\pi/6+\pi n), 1/(-\pi/6+\pi n)\) that lies in \([0,1]\)). Any open connected set containing \((0,0)\) would have to lie in the connected component \(\{0\}\times (-1/2,1/2)\text{;}\) but this is impossible since \(\{0\}\times (-1/2, 1/2)\) does not contain any open sets!
