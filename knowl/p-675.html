For each \(x\in X\) let \(C_x\) denote the (unique) connected component containing \(x\text{,}\) and let \(P_x\) denote the (unique) path connected component containing \(x\text{.}\) Let \(C\) be a connected component of \(X\text{.}\) For any \(x\in X\) we have \(P_x\cap C\ne\emptyset\text{;}\) since \(P_x\) is path connected, it is connected, and hence \(P_x\subseteq C\) by (1) above. It follows that \(C=\bigcup_{x\in C}P_x\) is a partition of \(C\) by path connected components.
