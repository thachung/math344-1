Theorem 1.23.1. Tietze extension theorem.

Let \(A\) be a closed subset of the normal space \(X\text{.}\)
  1. Any continuous function \(f\colon A\rightarrow [a,b]\) can be extended to a continuous function \(\widetilde{f}\colon X\rightarrow [a,b]\text{.}\)
  2. Any continuous function \(f\colon A\rightarrow \R\) can be extended to a continuous function \(\widetilde{f}\colon X\rightarrow \R\text{.}\)

Proof of (1).

Proof of (2).
