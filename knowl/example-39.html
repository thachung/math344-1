Example 1.9.5. Products and projections.

Let \(\{X_i, \mathcal{T}_i\}_{i\in I}\) be a collection of topological spaces, and let \(X=\prod_{i\in I}X_i\text{.}\) For each \(j\in I\) define the projection function \(pi_j\colon X\rightarrow X_j\) as \(\pi_j((x_i)_{i\in I})=x_i\text{.}\) In other words, the \(j\)-th projection map returns the \(j\)-th coordinate of an element of \(X\text{.}\)
  1. Prove: if \(X\) is given the product topology, then \(\pi_j\) is continuous for all \(j\in J\text{.}\)
  2. Prove: given a topology \(\mathcal{T}\) on \(X\text{,}\) we have \(\pi_j\) continuous for all \(j\in J\) if and only if \(\mathcal{T}\) contains the product topology. Thus, in particular, the product topology is the coarsest topology on \(X\) making each projection map continuous.
