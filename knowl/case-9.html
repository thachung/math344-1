Bijectivity.
To show \(f\) is bijective directly using Definition 0.1.23, we must show that \(f\) is injective and surjective. This is equivalent to showing that for \(y\in Y\) there exists a unique element \(x\in X\) such that \(f(x)=y\text{.}\)
Alternatively, using Theorem 0.1.27 we can show that \(f\) is bijective by providing an inverse function \(f^{-1}\colon Y\rightarrow X\text{.}\)
