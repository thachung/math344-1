    Section 1.2 Bases for topological spaces

    When defining a topology \(\mathcal{T}\) for a set \(X\) it is often convenient to specify instead a more manageable subset of \(\mathcal{T}\) that we can use to generate the entire topology using arbitrary unions and/or finite intersections (our open set operators). A topological basis is a particularly nice example of this (1.2.1) as the open sets of the topology it generates can be expressed as (arbitrary) unions of the basis elements. Bases will also provide a convenient way of comparing different topologies and (later on) determining whether relevant topological properties are satisfied by a given space.

    Definition 1.2.1. Topological basis.

    Let \(X\) be a set. A (topological) basis in \(X\) is a collection \(\mathcal{B}\) of subsets of \(X\) satisfying the following axioms:
    1. Covers.
      For each \(x\in X\) there is a set \(B\in \mathcal{B}\) such that \(x\in B\text{.}\)
    2. Contained in intersections.
      If \(B_1, B_2\in \mathcal{B}\text{,}\) then there is a \(B\in \mathcal{B}\) such that \(B\subseteq B_1\cap B_2\text{.}\)

    Example 1.2.2. Some bases for \(\R\).

    Show that the given collections \(\mathcal{B}\) are bases in \(\R\text{.}\)
    1. \(\displaystyle \mathcal{B}_1=\{(x-\epsilon, x+\epsilon)\colon x,\epsilon\in \R, \epsilon> 0\}\)
    2. \(\displaystyle \mathcal{B}_2=\{(a,b)\colon a,b\in \R, a< b\}\)
    3. \(\displaystyle \mathcal{B}_3=\{ (p,q)\colon p,q\in \Q, p< q\} \)
    4. \(\displaystyle \mathcal{B}_4=\{[a,b)\colon a,b\in \R, a< b\}\)
    Solution.

    Theorem 1.2.3.

    Let \(\mathcal{B}\subseteq \mathcal{P}(X)\) be a basis in the set \(X\text{.}\)
    1. The set
      \begin{equation*} \mathcal{T}=\{U\subseteq X\colon U=\bigcup_{i\in I}B_i \text{ for some elements } B_i\in \mathcal{B}\} \end{equation*}
      is a topology on \(X\text{.}\)
    2. We have the following alternate description of \(\mathcal{T}\text{:}\)
      \begin{equation*} U\in \mathcal{T} \iff \text{ for all } x\in U \ \text{ there exists } B\in \mathcal{B} \text{ such that } x\in B\subseteq U\text{.} \end{equation*}

    Definition 1.2.4.

    Let \(\mathcal{B}\subseteq \mathcal{P}(X)\) be a basis in the set \(X\text{.}\) The topology
    \begin{align} \mathcal{T}\amp =\{U\subseteq X\colon U=\bigcup_{i\in I}B_i \text{ for some elements } B_i\in \mathcal{B}\} \tag{1.2.1}\\ \amp = \{U\subseteq X\colon \text{ for all } x\in U \text{ there exists } B\in \mathcal{B} \text{ s.t. } x\in B\subseteq U\}\tag{1.2.2} \end{align}
    is called the topology generated by \(\mathcal{B}\text{.}\) Similarly, we call \(\mathcal{B}\) as basis for \(\mathcal{T}\text{.}\)

    Example 1.2.5. A basis for the standard topology on \(\R\).

    Observe that the topology \(\mathcal{T}\) generated by the basis \(\mathcal{B}_1\) in Example¬†1.2.2 is precisely the standard topology on \(\R\), as the description of \(\mathcal{T}\) in (1.2.2) makes clear.

    Theorem 1.2.6. Comparing with bases.

    Let \(\mathcal{T}\) be a topology on \(X\) generated by the basis \(\mathcal{B}\text{.}\)
    1. Let \(\mathcal{T}'\) be a topology on \(X\text{.}\) We have
      \begin{equation} \mathcal{T}\subseteq \mathcal{T}'\iff B\in \mathcal{T}' \text{ for all } B\in \mathcal{B}\text{.}\tag{1.2.3} \end{equation}
    2. Let \(\mathcal{T}'\) be a topology on \(X\) generated by the basis \(\mathcal{B}'\text{.}\) We have
      \begin{equation} \mathcal{T}\subseteq \mathcal{T}'\iff \text{ for all } B\in \mathcal{B} \text{ and } x\in B \text{ there exists } B'\in \mathcal{B}' \text{ s.t. } x\in B'\subseteq B\text{.}\tag{1.2.4} \end{equation}

    Example 1.2.7. Multiple bases for the standard real topology.

    Show that the bases \(\mathcal{B}_1, \mathcal{B}_2, \mathcal{B}_3\) from Example¬†1.2.2 all generate the standard topology on \(\R\text{.}\)
    Solution.

    Topological specimen 4.

    The collection \(\mathcal{B}_l=\{([a,b)\colon a,b\in \R, a< b\}\) is called the lower limit basis, and the topology \(\mathcal{T}_l\) it generates is called the lower limit topology on \(\R\text{.}\) We denote the topological space \((\R, \mathcal{T}_l)\) as \(\R_l\text{.}\)

    Example 1.2.8. Lower limit is finer than standard.

    Show that the lower limit topology on \(\R\) is strictly finer than the standard one.
    Solution.

    Topological specimen 5.

    Let \(K=\{1/n\colon c\in \Z_+\}=\{1, 1/2, 1/3,\dots\}\text{,}\) and define
    \begin{equation*} \mathcal{B}_K=\{(a,b)\colon a,b\in \R, a< b\}\cup \{ (a,b)-K\colon a,b\in \R, a< b\}\text{.} \end{equation*}
    The collection \(\mathcal{B}_K\) is a basis in \(\R\text{.}\) The topology \(\mathcal{T}_K\) it generates is called the \(K\)-topology on \(\R\text{.}\) We denote the topological space \((\R, \mathcal{T}_K)\) as \(\R_K\text{.}\)

    Proof.

    Example 1.2.9. Comparing standard, lower limit, and \(K\)-topology.

    Show that the \(K\)-topology is strictly finer than the standard topology on \(\R\text{.}\)
    Show that the lower limit topology and \(K\)-topology are incomparable.
    Solution.

    Theorem 1.2.10. Basis criterion.

    Let \(\mathcal{T}\) be a topology on the set \(X\text{.}\) A subset \(\mathcal{B}\subseteq \mathcal{T}\) is a basis of \(\mathcal{T}\) if and only if for all \(U\in \mathcal{T}\) and \(x\in U\) there is an element \(B\in \mathcal{B}\) such that \(x\in B\subseteq U\text{.}\)

    Proof.

    Example 1.2.11.

    Let \(X\) be a set, and let \(\mathcal{T}\) be the discrete topology on \(X\text{.}\) Show the following:
    1. \(\mathcal{B}=\{ \{x\}\colon x\in X\}\) is a basis for \(\mathcal{T}\text{.}\)
    2. If \(\mathcal{B}'\) is a basis for \(\mathcal{T}\text{,}\) then \(\mathcal{B}\subseteq \mathcal{B}'\text{.}\) In other words, \(\mathcal{B}\) is the minimal basis of the discrete topology.
    Solution.
