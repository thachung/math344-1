    Section 1.11 Quotients

    Topological specimen 13. Quotient space.

    Let \((X,\mathcal{T})\) be a topological space, \(\sim\) an equivalence relation on \(X\text{,}\) \(X^*=X/\sim\) the set of equivalence classes of \(\sim\text{,}\) and \(q\colon X\rightarrow X^*\) the map sending \(x\in X\) to its equivalence class \([x]_\sim\in X^*\text{.}\) The set
    \begin{equation*} \mathcal{T}_{\operatorname{quot}}=\{U\subset X^*\colon q^{-1}(U)\in \mathcal{T}\} \end{equation*}
    is a topology on \(X^*\) called the quotient topology. The map \(q\colon X\rightarrow X^*\) is a quotient map: that is, it is surjective and satisfies \(V\subseteq X^*\) is open if and only if \(q^{-1}(V)\) is open.

    Proof.

    That \(\mathcal{T}_{\operatorname{quot}}\) is a topology follows easily from properties of preimages of sets; and it satisfies the quotient map properties essentially by definition.

    Example 1.11.1.

    For each quotient space description give an explicit description of the equivalence classes and identify (without proof) what familiar space the quotient might be homeomorphic to.
    1. \(X=[0,1]\text{,}\) \(X^*\) obtained by identifying \(0\) and \(1\)
    2. \(X=\R\text{,}\) \(X^*=X/\sim\text{,}\) where \(x\sim y\) if and only if \(x=y+n\) for some \(n\in \Z\)
    3. \(X=[0,1]\times [0,1]\text{,}\) \(X^*\) obtained by identifying \(P=(0,y)\) and \(Q=(1,y)\) for all \(y\in [0,1]\)
    4. \(X=[0,1]\times [0,1]\text{,}\) \(X^*\) obtained by identifying \(P=(0,y)\) and \(Q=(1,y)\) for all \(y\in [0,1]\) and all \(R=(x,0)\) and \(S=(x,1)\) for \(x\in [0,1]\text{.}\)
    Solution.

    Definition 1.11.2.

    Let \(X\) and \(Y\) be topological spaces. A function \(f\colon X\rightarrow Y\) is a quotient map if it satisfies the following properties:
    1. \(f\) is surjective;
    2. \(V\subseteq Y\) is open if and only if \(f^{-1}(V)\subseteq X\) is open.
    It follows immediately from the definition that a quotient map is continuous.

    Definition 1.11.3. Saturated sets.

    Let \(f\colon X\rightarrow Y\text{.}\) A set \(A\subseteq X\) is saturated with respect to \(f\) if \(f^{-1}(f(A))=A\text{.}\) Equivalently, \(A\) is saturated if if whenever \(x\in A\) we have \(f^{-1}(\{f(x)\})\subseteq A\text{.}\)

    Theorem 1.11.4. Quotient map properties.

    1. Let \(f\colon X\rightarrow Y\) be a continuous surjective function. The following statements are equivalent.
      1. \(f\) is a quotient map.
      2. \(f(U)\) is open for all open saturated sets \(U\subseteq X\text{.}\)
      3. \(f(C)\) is closed for all closed saturated subsets \(C\subseteq X\text{.}\)
    2. Let \(f\) be continuous and surjective. If \(f\) is open or closed, then it is a quotient map.
    3. The composition of two quotient maps is a quotient map.
    Statement (3) is straightforward. Statement (2) follows easily from (1) using equivalent statements (b) and (c), respectively.

    Proof of (1).

    We prove \((a)\iff (b)\) and \((b)\iff (c)\text{.}\)
    \((a)\implies (b)\text{.}\) Assume \(f\) is a quotient map. Let \(U\subseteq X\) be open and saturated. Since \(f\) is a quotient map, to show \(V=f(U)\) is open, it is enough to show \(f^{-1}(V)=f^{-1}(f(U))\) is open. I claim \(f^{-1}(f(U))=U\text{,}\) whence the claim follows. In general for any function of sets we have \(f^{-1}(f(U))\supseteq U\text{.}\) (Check for yourself.) Since \(U\) is saturated, the reverse inclusion also holds. Indeed, if \(x\in f^{-1}(f(U))\text{,}\) then \(f(x)\in f(U)\) and thus \(f(x)=f(u)\) for some \(u\in U\text{;}\) since \(U\) is saturated, we have \(x\in U\text{.}\)
    \((b)\implies (a)\text{.}\) Assume \(U=f^{-1}(V)\) be open for \(V\subseteq Y\text{.}\) We wish to show that \(V\) is open in \(Y\text{.}\) First observe that \(f^{-1}(V)\) is saturated: if \(x\in f^{-1}(V)\text{,}\) then \(f^{-1}(\{f(x)\})\subseteq f^{-1}(V)\text{.}\) Since \(f\) satisfies \((b)\) and \(U\) is an open saturated set, we have \(f(U)\) open. I claim \(f(U)=f(f^{-1}(V))=V\text{,}\) showing \(V\) is open. In general we always have \(f(f^{-1}(V))\subseteq V\) for any function of sets. For the other inclusion, given \(y\in V\text{,}\) there is an \(x\in X\) such that \(f(x)=y\text{;}\) then \(x\in f^{-1}(V)\) and \(f(x)=y\text{,}\) showing \(y\in f(f^{-1}(V))\text{.}\)
    \((b)\implies (c)\text{.}\) Let \(C\) be a saturated closed map. First I claim that the open set \(U=X-C\) is saturated. To see why, suppose \(y=f(x)\) for some \(x\in U\text{.}\) Since \(x\notin C\) and \(C\) is saturated, we must have \(f^{-1}(\{y\})\cap C=\emptyset\text{.}\) Thus \(f^{-1}(\{y\})\subseteq U\text{,}\) as desired. Next, I claim \(f(U)=Y-f(C)\text{.}\) We have already seen that if \(y\in f(U)\text{,}\) then \(f^{-1}(\{y\})\cap C=\emptyset\text{,}\) and hence \(y\notin f(C)\text{.}\) This proves \(f(U)\subseteq Y-f(C)\text{.}\) For the other inclusion, given any \(y\in Y-f(C)\text{,}\) by surjectivity there is some \(x\in X\) such that \(f(x)=y\text{;}\) since \(y\notin f(C)\) we must have \(x\in U\text{;}\) thus \(y\in f(U)\text{.}\) This proves \(f(U)=X-f(C)\text{.}\) Since \(f\) satisfies (b), \(f(U)\) is open. We conclude that \(f(C)\) is closed, as desired.
    \((c)\implies (b)\text{.}\) This argument is essentially identical to the one above.

    Theorem 1.11.5. Mapping property of quotient maps.

    Let \(p\colon X\rightarrow Y\) be a quotient map. Given any continuous function \(g\colon X\rightarrow Z\) satisfying
    \begin{equation} p(x)=p(x') \implies g(x)=g(x')\tag{1.11.1} \end{equation}
    for all \(x, x'\in X\) there is a unique continuous function \(f\colon Y\rightarrow Z\) satisfying \(g=f\circ p\text{.}\)

    Proof.

    Assume we have \(g\colon X\rightarrow Z\) as described. Define \(f\colon Y\rightarrow Z\) via the following recipe: given \(y\in Y\text{,}\) pick any \(x\in X\) such that \(p(x)=y\) (possible since \(p\) is surjective); now define \(f(y)=g(x)\text{.}\) Two comments:
    1. To make sure \(f\) as described is well-defined, we must make sure that the definition of \(f(y)\) does not depend on the choice of \(x\) involve: i.e., if you happen to pick another \(x'\) such that \(p(x')=y\text{,}\) we must make sure that \(g(x)=g(x')\text{.}\) This is precisely what is guaranteed by the condition that \(p(x)=p(x')\implies g(x)=g(x')\text{.}\)
    2. Since we want \(g=f\circ p\text{,}\) this definition of \(f\) is forced upon us: given \(y=p(x)\text{,}\) we must have \(f(y)=f(p(x))=g(x)\text{.}\) Thus \(f\) is unique.
    Now we must show \(f\) is continuous. We must show, given any open \(V\subseteq Z\) the set \(U=f^{-1}(V)\) is open. To do so we use the fact that \(p\) is a quotient map: that is, we need only show that \(p^{-1}(U)\) is open in \(X\text{.}\) But we have
    \begin{equation*} p^{-1}(U)=p^{-1}(f^{-1}(V))=(f\circ p)^{-1}(V)=g^{-1}(V), \end{equation*}
    since \(g=f\circ p\text{.}\) Since \(g\) is continuous, we conclude that \(p^{-1}(U)=g^{-1}(V)\) is open, as desired.
    Lastly, we look at when \(f\) is a quotient map. If \(f\) is a quotient map, then so is the composition \(g=f\circ p\) by statement (3) of Theorem¬†1.11.4. Assume \(g\) is a quotient map. Since \(g\) is surjective and \(g=f\circ p\text{,}\) it follows that \(f\) must be surjective. Now suppose \(U=f^{-1}(V)\) is open in \(Y\text{.}\) Since \(p\) is continuous, \(p^{-1}(U)\) is open in \(X\text{.}\) As we saw above, \(p^{-1}(U)=g^{-1}(V)\text{,}\) which is a saturated open set of \(X\text{.}\) Since \(g\) is a quotient map, the set \(g(g^{-1}(V))\) is open. But again, because \(g\) is surejective, we have \(g(g^{-1}(V))=V\text{.}\)

    Definition 1.11.6. Fibers of maps.

    For any function \(f\colon X\rightarrow Y\) between sets and element \(y\in Y\text{,}\) we call \(f^{-1}(\{y\})\) the fiber of \(f\) lying over \(y\). We say a function \(g\colon X\rightarrow Z\) agrees on fibers of \(f\) if
    \begin{equation*} f(x)=f(x')\implies g(x)=g(x')\text{.} \end{equation*}

    Corollary 1.11.7. Mapping property of quotients.

    Let \(X^*\) be the quotient space of \(X\) by the equivalence relation \(\sim\text{,}\) and let \(q\colon X\rightarrow X^*\) be the corresponding quotient map. Let \(g\colon X\rightarrow Z\) be a continuous function that agrees on fibers of \(q\) (i.e., \(x\sim x'\implies g(x)=g(x')\)).
    1. There is a unique continuous function \(f\) satisfying \(g=f\circ q\text{;}\) and \(f\) is a quotient map if and only if \(g\) is a quotient map.
    2. The function \(f\) is a homeomorphism if and only if \(g\) is a quotient map and satisfies
      \begin{equation*} g(x)=g(x') \iff x\sim x' (\text{equivalently, } g(x)=g(x')\iff q(x)=q(x')) \end{equation*}
      for all \(x, x'\in X\text{.}\)

    Proof.

    1. The statement is a direct consequence of Theorem¬†1.11.5 by setting \(Y=X^*\) and \(p=q\text{,}\) and understanding that the implication \(x\sim x'\implies g(x)=g(x')\) is equivalent to \(p(x)=p(x')\implies g(x)=g(x')\text{.}\)
    2. The condition \(x\sim x' \iff g(x)=g(x')\) guarantees that the function \(f\colon X^*\rightarrow Y\) is injective. The condition of \(g\) being a quotient map is equivalent to \(f\) being a quotient map according to Theorem¬†1.11.5. Thus the two conditions taken together are equivalent to \(f\) being a bijective quotient map. This in turn is equivalent to being a homeomorphism by equivalent statement 1.b of Theorem¬†1.9.7, since all subsets of \(X\) are saturated with respect to an invertible function \(f\text{.}\)

    Corollary 1.11.8. Factoring maps through a quotient.

    Let \(g\colon X\rightarrow Y\) be a surjective continuous map, and let \(X^*\) be the quotient obtained from \(X\) by treating each fiber \(g^{-1}(\{y\})\text{,}\) \(y\in Y\text{,}\) as a single point: i.e., the equivalence relation is \(x\sim x'\iff g(x)=g(x')\text{.}\)
    1. There is a unique bijective continuous function \(f\colon X^*\rightarrow Y\) satisfying \(g=f\circ q\text{.}\)
    2. The function \(f\) is a homeomorphism if and only if \(g\) is a quotient map.
    3. If \(Y\) is Hausdorff, then \(X^*\) is Hausdorff.

    Proof.

    With the exception of (3), this corollary follows essentially directly from Corollary¬†1.11.7. Statement (3) follows directly from (1). In fact, more generally, if \(f\colon X\rightarrow Y\) is a bijective continuous map, and \(Y\) is Hausdorff, then \(X\) is Hausdorff.

    Example 1.11.9. The circle as a quotient space.

    Let \(X^*\) denote the quotient obtained from \(X=[0,1]\) after identifying \(0\) and \(1\text{,}\) and let \(\R/\Z\) denote the quotient of \(\R\) by the equivalence relation \(x\sim y\) if and only if \(x=y+n\) for some \(n\in Z\text{.}\) Prove that \(\R/\Z\) is homeomorphic to \(S^1=\{(x,y)\colon x^2+y^2=1\}\text{.}\)
    1. Prove: \(X^*\cong \R/\Z\)
    2. Prove: \(X^*\cong \R/\Z\cong S^1\text{.}\)
    Note: once we have some notions about compactness under our belt it will be easy to show directly that \(X^*\cong S^1\text{.}\) Part (2) is a workaround for the time being.
    Solution.
    1. Consider the map \(g\) defined by composition
      where \(q\) is the quotient map from \(\R\) to \(\R/\Z\text{.}\) It is easy to see that \(g\) is surjective and satisfies \(g(t)=g(t')\) if and only if \(t=t'\) or \(t,t'\in \{0,1\}\text{.}\) From Corollary¬†1.11.7 we conclude that \(g\) gives rise to a bijective continuous function \(f\colon X^*\rightarrow \R/\Z\text{.}\) Concretely, \(f\) takes an equivalence class \([t]\in X^*\) and maps it to the equivalence class \(\{t+n\colon n\in \Z\}\) in \(\R/\Z\text{.}\) Furthermore, to show \(f\) is a homeomorphism it is enough to prove that \(g\colon [0,1]\rightarrow \R/\Z\) is a quotient map. In fact, we can show that \(g\) is a closed map. Take a closed set \(C\subseteq [0,1]\text{.}\) To show its image \(g(C)\) is closed in \(\R/Z\text{,}\) we need only show, using the fact that \(q\) is a quotient map, that \(q^{-1}(g(C))\) is closed in \(\R/\Z\text{.}\) We have \(q^{-1}(g(C))=C+\Z=\{c+n\colon c\in C, n\in \Z\}\text{.}\) This set is the result of starting with our original closed set \(C\subseteq [0,1]\) and taking the union of translated copies of the form \(C+n\subseteq [n,n+1]\text{,}\) where \(n\in \Z\text{.}\) It is easy to see that each translate \(C+n\) is closed; since furthermore \(C+n\subseteq [n,n+1]\text{,}\) it follows that the union is also closed. I'll leave the details to you.
    2. Let \(q\colon\R\rightarrow \R/\Z\) be the quotient map. Consider the diagram
      where \(g(t)=(\cos 2\pi t, \sin 2\pi t)\text{.}\) It is easy to see that \(g\) is continuous, surjective, and satisfies \(g(t_1)=g(t_2)\iff t_1=t_2+n\) for some \(n\in \Z\text{.}\) It follows from Corollary¬†1.11.7 that the map \(f\) is bijective and continuous. Furthermore, by the same result, we can show \(f\) is a homeomorphism as long as we can show \(g\) is a quotient map. I'll show that in fact \(g\colon \R\rightarrow S^1\) is open, and hence a projection map.
      To this end let \(U\subseteq \R\) be any open set, and let \(P\in f(U)\text{,}\) so that \(P=g(t)\) for some \(t\in U\text{.}\) We will find an open interval \(I\) satisfying \(t\in I\subseteq U\) such that \(f(I)\) is open. Since \(P\in f(I)\subseteq f(U)\) and \(P\in f(U)\) was arbitrary, this will show that \(f(U)\) is open.
      Let \(g(t)=P=(x,y)\text{.}\) Since we cannot have \(x=y=0\text{,}\) we have four cases: \(y>0\) or \(y< 0\) or \(x> 0\) or \(x< 0\text{.}\) I will treat the first case; my argument is easily adapted to the other cases. Assume \(y> 0\text{.}\) Since the set \(V=S^1\cap \left(\R\times (0,\infty)\right)\) is open in \(S^1\text{,}\) we can find an open interval \(I=(a,b)\ni t\) contained in \(U\) such \(g(I)\subseteq V \text{:}\) i.e., all points in \(g(I)\) lie in the the upper half plane. Elementary trigonometric properties tell us that the function \(h(t)=\cos 2\pi t\) is strictly decreasing on \(I\text{,}\) and hence that \(h(I)\) is the open interval \((\cos 2\pi b, \cos 2\pi a)=(c,d)\text{.}\) It is now easy to see that \(g(I)=\{(x,y)\colon x\in (c,d), y=+\sqrt{1-x^2}\}=S^1\cap \left((c,d)\times \R\right)\text{,}\) which is open.
