    Section 1.24 Nets

    Definition 1.24.1. Partial ordering axioms.

    Let \(R\) denote a binary relation on the set \(I\text{.}\) We write \(iRj\) to assert that the relation \(R\) holds between elements \(i,j\in I\text{.}\) The relation \(R\) is a partial ordering if it satisfies the following three axioms.
    • Reflexive (PO1).
      For all \(i\in I\text{,}\) we have \(iRi\text{.}\)
    • Transitive (PO2).
      For all \(i,j,k\in I\text{,}\) if \(iRj\) and \(jRk\text{,}\) then \(iRk\text{.}\)
    • Antisymmetric (PO3).
      For all \(i,j\in I\text{,}\) if \(iRj\) and \(jRi\text{,}\) then \(i=j\text{.}\)
    The relation \(R\) is a quasiordering (or preordering) if it satisfies axioms PO1 and PO2. When \(R\) is a partial ordering or a quasiordering, we will often write \(i\leq j\) for \(iRj\text{.}\)
    A partially ordered set (POSET, for short) is a pair \((I,\leq)\text{,}\) where \(\leq\) is a partial ordering on \(I\text{.}\) Similarly,a quasiordered set is a pair \((I,\leq)\text{,}\) where \(\leq\) is a a quasiordering on \(I\text{.}\)

    Example 1.24.2. Common examples.

    1. The usual ‚Äúless than or equal to‚Äù relation \(x\leq y\) is a partial ordering on \(\R\text{.}\)
    2. Let \(X\) be a set, and let \(I=\mathcal{P}(X)\) be the power set of of \(X\text{.}\) The relation \(A\leq B \iff A\subseteq B\) defines a partial ordering on \(I\text{.}\)
    3. Let \(X\) be a set, and let \(I=\mathcal{P}(X)\) be the power set of of \(X\text{.}\) The relation \(A\leq B \iff A\supseteq B\) also defines a partial ordering on \(I\text{.}\)
    4. For an element \(x\) in a topological space \(X\) the set \(I\) of all open neighborhoods of \(x\) is a partially ordered set under reverse inclusion: i.e., \(U\leq V\iff U\supseteq V\text{.}\)
    5. Recall that an integer \(m\) divides an integer \(n\text{,}\) written \(m\mid n\text{,}\) if there is an integer \(q\) such that \(n=mq\text{.}\) The relation \(m\leq n\iff m\mid n\) defines a quasiordering on \(\Z\text{,}\) but not a partial ordering: \(2\mid -2\) and \(-2\mid 2\text{,}\) but \(2\ne -2\text{.}\) The same relation does define a partial ordering on \(\Z_+\text{.}\)

    Definition 1.24.3. Directed set.

    A directed set is a pair \((I,\leq)\text{,}\) where \(\leq\) is a quasiordering on \(I\) that satisfies the following axiom.
    • Directed (D).
      For all \(i,j\in I\text{,}\) there is a \(k\in I\) such that \(i\leq k\) and \(j\leq k\text{.}\)
    A subset \(J\) of a directed set \(I\) is cofinal if for all \(i\in I\) there is a \(j\geq i\) such that \(j\in J\text{.}\)

    Example 1.24.4. Common examples.

    Each quasiordered (and/or partially ordered) set in Example¬†1.24.2 is easily seen to satisfy the further axiom (D), and is thus a directed set.

    Definition 1.24.5. Nets and convergent nets.

    Let \(X\) be a set. A net in \(X\) is a function \(f\colon I\rightarrow X\) where \(I\) is a directed set. In particular, a net is a tuple, and accordingly we may use the coordinate notation \(f=(x_i)_{i\in I}\text{,}\) where \(x_i=f(i)\text{.}\)
    A net \(f\colon I\rightarrow X\) is eventually in a subset \(A\subseteq X\) if there is an element \(i_0\in I\) such that \(x_j\in A\) for all \(j\geq i_0\text{.}\)
    Assume \(X\) is a topological space. A net \(f\colon I\rightarrow X\) converges to an element \(x\in X\text{,}\) denoted \(x_i\rightarrow x\text{,}\) if for all open sets \(U\) containing \(x\) the net \(f=(x_i)_{i\in I}\) is eventually in \(U\text{.}\)
    A net \(f\colon I\rightarrow X\) is cofinally in a subset \(A\subseteq X\) if the set \(J=f^{-1}(A)\) is cofinal: equivalently, if for all \(i\in I\) there is a \(j\geq i\) such that \(x_j=f(j)\in A\text{.}\)
    An element \(x\in X\) is a limit point of the net \(f\colon I\rightarrow X\) if \(f=(x_i)\) is cofinally in every open neighborhood of \(x\text{.}\)

    Example 1.24.6. Riemann integral.

    A careful examination of the Riemann integral of a function over an interval \([a,b]\) reveals that it is a statement about a convergent net. Indeed, let \(I\) be the set of all ‚Äúpointed partitions‚Äù of \([a,b]\) into subintervals: i.e., an element of \(I\) is a pair \((\mathcal{P}, \boldx)\text{,}\) where \(\mathcal{P}\) is a subinterval partition
    \begin{equation*} \mathcal{P}\colon a=x_0 < x_1< \dots < x_n=b \end{equation*}
    and \(\boldx=(x_1^*, x_2^*,\dots, x_n^*)\) is a choice of ‚Äúsample points‚Äù in each subinterval of \(\mathcal{P}\text{.}\) The relation \((\mathcal{P},\boldx)\leq (\mathcal{P}', \boldx')\iff \mathcal{P}\subseteq \mathcal{P}'\) is a quasiordering on \(I\) satisfying (D). To any function \(f\colon [a,b]\rightarrow \R\) we can associate the net \((R_i)_{i\in I}\text{,}\) where for each \(i=(\mathcal{P}, \boldx)\in I\) we define \(R_i\) to be the corresponding Riemann sum of \(f\text{.}\) The Riemann integral \(\int_a^b f\, dx\) exists and is equal to \(R\) if and only the net \((R_i)_{i\in I}\) converges to \(R\text{.}\)

    Theorem 1.24.7. Nets and topology.

    Let \(X\) be a topological space.
    1. The closure \(\overline{A}\) of any set \(A\) is the set of all \(x\in X\) for which there is a net \(f\colon I\rightarrow A\) such that \(x_i\rightarrow x\text{.}\)
      It follows that a subset \(A\) is closed if and only if it contains all limits of all convergent nets in \(A\text{.}\)
    2. A function \(f\colon X\rightarrow Y\) is continuous if and only if for all convergent nets \(h\colon I\rightarrow X\) the net \(f\circ h\colon I\rightarrow Y\) is convergent.
    3. The space \(X\) is Hausdorff if and only if every net in \(X\) converges to at most one element in \(X\text{.}\)

    Proof.

    1. Assume \(x\in \overline{A}\text{.}\) For all open sets \(U\) containing \(x\text{,}\) choose \(x_U\in U\cap A\text{.}\) Let \(I\) be the set of all open neighborhoods of \(x\text{,}\) considered as a directed set with reverse inclusion. Define \(f\colon I\rightarrow A\) as \(f(U)=x_U\text{.}\) The net \((x_U)_{U\in I}\) converges to \(x\text{.}\) Indeed, for any open set \(V\) containing \(x\text{,}\) we set \(i_0=V\in I\text{:}\) if \(U\geq V\text{,}\) then \(U\subseteq V\) (by definition of our ordering), and hence \(x_U\in U\subseteq V\text{.}\)
      Assume \(x_i\rightarrow x\) for some net \((x_i)_{i\in I}\) in \(A\text{.}\) Given any open set \(U\) containing \(x\text{,}\) there is an \(i_0\in I\) such that \(x_j\in U\) for all \(j\geq i_0\text{.}\) In particular, we have \(x_{i_0}\in U\text{.}\) Since \((x_i)_{i\in I}\) is a net in \(A\text{,}\) we conclude that \(x_{i_0}\in\cap A\cap U\text{.}\)
      Note: the argument for the forward direction uses the axiom of choice potentially. If that bugs you, consider the following alternative: let \(I=\{(U,y)\colon U\text{ open neighborhood of } x, y\in U\cap A\}\) with quasiordering \((U,y)\leq (U', y')\iff U\supseteq U'\text{.}\) (This is no longer a partial ordering necessarily.) Define \(f\colon I\rightarrow X\) as \(f((U,y))=y\text{.}\) This net also converges to \(x\text{.}\)
    2. Assume \(f\) is continuous and \((x_i)_{i\in I}\) is a net satisfying \(x_i\rightarrow x\text{.}\) Given any open set \(U\) containing \(y\text{,}\) the net \((x_i)_{i\in I}\) is eventually in \(f^{-1}(U)\text{,}\) since this is an open set containing \(x\text{.}\) But then clearly \((f(x_i))_{i\in I}\) is eventually in \(f(f^{-1}(U))\subseteq U\text{.}\)
      Inversely, if \(f\) is not continuous, then there is an \(x\in X\) and open set \(U\) containing \(y=f(x)\) such that for all open sets \(V\) containing \(x\text{,}\) there is an element \(y_V=f(x_V)\in f(V)\cap (Y-U)\text{.}\) Let \(I\) be the set of all open neighborhoods of \(x\) with partial ordering given by reverse inclusion, and consider the net \((x_V)_{V\in I}\text{,}\) where \(x_V\) is the element described above. We have \(x_V\rightarrow x\text{,}\) using a similar argument as in (1), and yet \((f(x_V))_{V\in I}=(y_V)_{V\in I}\not\rightarrow y\text{.}\) Indeed, by construction the net \((y_V)_{V\in I}\) never lies in the open neighoborhood \(U\) of \(y\text{.}\)

    Definition 1.24.8. Subnets.

    Let \((J,\leq_J)\) and \((I,\leq_I)\) be quasiordered sets. An order-preserving map is a function \(g\colon J\rightarrow I\) satisfying the following property: if \(j\leq_J j'\text{,}\) then \(g(j)\leq_I g(j')\text{.}\)
    Let \(f\colon I\rightarrow X\) be a net. A subnet of \(I\) is a net of the form \(f\circ g\colon J\rightarrow X\text{,}\) where \(J\) is a directed set, and \(g\colon J\rightarrow I\) is an order-preserving map, and the image \(g(J)\) is cofinal in \(I\text{.}\)

    Theorem 1.24.9. Limit points of nets.

    Let \(X\) be a topological space, and let \(f=(x_i)_{i\in I}\) be a net in \(X\text{.}\) An element \(x\in X\) is a limit point of \((x_i)_{i\in I}\) if and only if there is a subnet of \((x_i)_{i\in I}\) converging to \(x\text{.}\)

    Lemma 1.24.10. Kelley's lemma.

    Let \(f\colon I\rightarrow X\) be a net. If \(\mathcal{A}\) is a collection of subsets of \(X\) satisfying the conditions
    1. for all \(A\in \mathcal{A}\) the set \(I_A=f^{-1}(A)\) is cofinal,
    2. for all \(A_1, A_2\in \mathcal{A}\) there is an \(A_3\in \mathcal{A}\) such that \(A'\subseteq A_1\cap A_2\text{,}\)
    then there is a subnet of \(f=(x_i)_{i\in I}\) which is eventually in \(A\) for all \(A\in\mathcal{A}\text{.}\)
